DECISÕES ARQUITETURAIS CRÍTICAS (ADRs)
ADR-001: SSG vs SSR vs CSR para páginas de serviço
Decisão:
SSG (Static Site Generation) com ISR (Incremental Static Regeneration)
Contexto:
Páginas de serviço mudam pouco (1x/mês)
SEO é crítico (Google prioriza LCP < 2.5s)
1000s de visitas/dia (custo de SSR seria alto)
Implementação:
Consequências:
✅ LCP < 1s (Google loves)
✅ Custo zero (Vercel serve do CDN)
❌ Updates demoram até 1h (aceitável)
ADR-002: Chatbot IA vs Chatbot baseado em regras
Decisão:
Híbrido (regras + IA fallback)
Contexto:
Perguntas comuns (90%) são previsíveis ("Quanto custa PGR?")
Perguntas complexas (10%) precisam de IA
Implementação:

Escudo Site: Porque seu site deveria vender enquanto você dorme.

Construir o site do zero com um nível de acabamento premium — tipografia refinada, espaçamento generoso, hero com imagem profissional, micro-interações sofisticadas e layout que transmite autoridade técnica real.

O projeto deve estar em português do Brasil.

Usar a paleta de cor abaixo no projeto:

Verde #2E7D32 é um excelente ponto de partida. Ele comunica conformidade, saúde, sustentabilidade e “está tudo sob controle”. Mas um sistema de cores não vive de uma cor só. Precisamos construir um ecossistema em torno dela — como um bom sistema operacional, não apenas um app isolado.

Vamos arquitetar isso.

Cor base — Verde Segurança
#2E7D32
Uso: botões primários, indicadores positivos, ícones de conformidade, destaques estratégicos.

Agora criamos variações tonais para hierarquia visual:

Verde escuro (autoridade técnica)
#1B5E20
Para header, footer, áreas institucionais. Passa robustez e maturidade.

Verde médio (variação para hover)
#256628
Para estados ativos e hover de botão.

Verde claro (fundos suaves e alertas positivos)
#A5D6A7
Ideal para caixas informativas, sucesso, banners leves.

Verde muito claro (background suave)
#E8F5E9
Cria sensação de organização e higiene visual.

Agora os neutros estruturais — fundamentais para evitar “site monocromático de clínica”:

Cinza grafite técnico
#263238
Texto principal. Forte contraste, profissional.

Cinza claro industrial
#CFD8DC
Divisores, bordas, tabelas.

Branco puro
#FFFFFF
Respiro. Segurança também é clareza.

Cor de contraste estratégica (porque verde puro demais vira monotonia):

Laranja industrial suave
#F9A825
Para CTA principal (ex: “Solicitar Laudo”, “Agendar Inspeção”).
Esse contraste verde + laranja ativa atenção sem cair no vermelho alarmista.

E para estados críticos:

Vermelho técnico
#C62828
Somente para risco real, não para decoração.

Agora o raciocínio arquitetural:

– 60% neutros claros
– 30% verde em suas variações
– 10% cor de contraste (laranja ou vermelho)

Isso mantém equilíbrio cognitivo. Verde em excesso pode parecer ambientalista; em proporção correta, comunica responsabilidade técnica.

Se você quiser levar isso para um visual mais tecnológico (SaaS de gestão de EPI, dashboards, IoT), podemos escurecer o fundo e usar o verde como cor de “status ativo”, quase como LEDs industriais digitais.

Cor não é estética. É semântica visual codificada.
