# Checklist de Publicação do Blog

Use este checklist antes de publicar qualquer artigo novo.

## Conteúdo

- [ ] O tema está claro
- [ ] O título está forte e objetivo
- [ ] O texto tem introdução, desenvolvimento e conclusão
- [ ] Os subtítulos estão organizados
- [ ] O artigo está escrito em linguagem clara
- [ ] As siglas foram explicadas na primeira menção
- [ ] O texto foi revisado ortograficamente

## Metadados

- [ ] O slug é único
- [ ] O excerpt está curto e útil
- [ ] A categoria está correta
- [ ] A data de publicação está correta
- [ ] O tempo de leitura faz sentido
- [ ] O metaTitle está consistente com o título
- [ ] O metaDescription está claro e atrativo

## Imagem

- [ ] A imagem de capa existe no projeto
- [ ] O caminho de coverImage está correto
- [ ] A imagem combina com o tema do artigo
- [ ] A imagem não está quebrada no navegador

## Autoria

- [ ] Se o artigo for meu, author está como `Tassia dos Santos`
- [ ] Se não for meu, o author foi ajustado para o nome correto
- [ ] Não há assinatura duplicada no corpo do artigo

## Testes

- [ ] O artigo abre no navegador
- [ ] O card aparece na listagem do blog
- [ ] A imagem de capa carrega
- [ ] A formatação Markdown está correta
- [ ] O build do frontend passou sem erro

## Comandos úteis

```sh
cd artifacts/sst-consultoria
pnpm dev
pnpm build
```
