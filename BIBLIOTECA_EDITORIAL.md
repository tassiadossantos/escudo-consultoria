## Como usar

1. Comece pelo [Fluxo Editorial Ideal](FLUXO_EDITORIAL_BLOG.md) para entender o processo.
2. Use o [Modelo Pronto de Artigo Novo](MODELO_ARTIGO_BLOG.md) ou o [Template de Markdown](TEMPLATE_MARKDOWN_ARTIGO_BLOG.md) para escrever.
3. Consulte a [Formatação Premium do Artigo](FORMATO_PREMIUM_ARTIGO_BLOG.md) para manter o padrão visual.
4. Antes de publicar, valide tudo com o [Checklist de Publicação](CHECKLIST_PUBLICACAO_BLOG.md).
5. Se precisar de orientação geral, volte ao [Guia para Criar Novos Artigos do Blog](GUIA_ARTIGOS_BLOG.md).

## Regra prática

- Se o artigo for seu, use `author: "Tassia dos Santos"`.
- Se não for seu, troque o autor para o nome correto.
- Não duplique a assinatura no corpo do texto se o componente já a renderiza automaticamente.
- Sempre teste no navegador e rode o build antes de publicar.
