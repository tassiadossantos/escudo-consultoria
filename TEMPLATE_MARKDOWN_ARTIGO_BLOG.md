# Template de Markdown para Novo Artigo

Copie este arquivo e preencha os campos.

```md
---
title: "Título do artigo"
slug: "slug-unico-do-artigo"
excerpt: "Resumo curto e objetivo do artigo."
coverImage: "/images/blog/minha-imagem.jpg"
author: "Tassia dos Santos"
publishedAt: "2026-04-01"
category: "Categoria"
readingTime: 8
metaTitle: "Título SEO do artigo"
metaDescription: "Descrição SEO curta e clara."
---

# Título do artigo

Introdução do artigo.

## Seção 1

Explique o problema, contexto ou cenário.

## Seção 2

Apresente a parte técnica, prática ou educativa.

### Subtópico opcional

Use este espaço para detalhar um ponto específico.

## Seção 3

Mostre consequências, erros comuns ou recomendações.

## Conclusão

Feche com a mensagem principal e uma chamada final.
```

## Observações

- Troque `author` se o texto não for seu.
- Use um `slug` único.
- Verifique se a imagem de capa existe.
- Mantenha parágrafos curtos para leitura fácil.
